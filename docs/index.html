Shifty

A JavaScript library to add parallax effects to background-image

Download GitHub

Quick start

Add the following description in the <head> tag of the web page you want to use Shifty.

You can also bundle the library with Browserify/Webpack

import Shifty from 'shifty'

Just replace the style="background-image:url(...)" to data-img="..."

<div class="js-parallax"
  data-img="https://source.unsplash.com/pb_lF8VWaPU/1600x900">
  <p>contents here</p>
</div>

Write the following script

new Shifty('.js-scrollable');
